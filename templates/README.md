# DevTeam6 Codespace Templates

Complete, production-ready starter templates for React, Vue 3, and .NET 8 development.

## 📂 Available Templates

### 🔵 React Starter (`react-starter/`)
- **React 18** + TypeScript + Vite
- ESLint & Prettier configured
- Sample Button component
- Hot Module Replacement (HMR)
- **Port**: 3000

```bash
cd templates/react-starter
npm install
npm run dev
```

### 🟢 Vue 3 Starter (`vue3-starter/`)
- **Vue 3** Composition API + TypeScript + Vite
- Vue Router for navigation
- Pinia for state management
- ESLint & Prettier configured
- Example views and components
- **Port**: 5173

```bash
cd templates/vue3-starter
npm install
npm run dev
```

### 🟣 .NET 8 API (`dotnet8-api/`)
- **ASP.NET Core 8** Web API
- Swagger/OpenAPI documentation
- CORS enabled
- Example WeatherForecast controller
- Logging configured
- **Port**: 5000

```bash
cd templates/dotnet8-api
dotnet restore
dotnet run
```
