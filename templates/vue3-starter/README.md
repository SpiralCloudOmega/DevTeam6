## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5174`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

### Linting

```bash
npm run lint
```

### Format Code

```bash
npm run format
```

## Customization

This template is designed to be a starting point. Feel free to:

- Add UI component libraries like [Vuetify](https://vuetifyjs.com/) or [Element Plus](https://element-plus.org/)
- Add form validation with [VeeValidate](https://vee-validate.logaretm.com/)
- Add HTTP client like [Axios](https://axios-http.com/)
- Add testing with [Vitest](https://vitest.dev/) and [Vue Test Utils](https://test-utils.vuejs.org/)
- Add styling solutions like [Tailwind CSS](https://tailwindcss.com/) or [UnoCSS](https://unocss.dev/)
