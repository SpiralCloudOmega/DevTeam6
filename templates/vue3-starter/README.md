# Vue 3 + TypeScript + Vite Starter

This template provides a complete setup for Vue 3 development with TypeScript, Vue Router, and Pinia state management.

## Getting Started

```bash
npm install
npm run dev
```

## Features

- ⚡️ Vue 3 with Composition API
- 🛠️ TypeScript
- 🔥 Vite for fast development
- 🚦 Vue Router for navigation
- 📦 Pinia for state management
- 📝 ESLint configured

## Available Scripts

- `npm run dev` - Start development server on port 5173
- `npm run build` - Build for production with type checking
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Project Structure

```
src/
├── assets/         # Static assets
├── components/     # Vue components
├── views/          # Route views
├── router/         # Vue Router configuration
├── stores/         # Pinia stores
├── App.vue         # Main app component
├── main.ts         # Entry point
└── style.css       # Global styles
```

## State Management

This template includes Pinia with a counter store example. Check `src/stores/counter.ts` for details.

## Routing

Vue Router is configured with two example routes. Check `src/router/index.ts` for configuration.
