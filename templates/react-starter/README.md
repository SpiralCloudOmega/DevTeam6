## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

### Linting

```bash
npm run lint
```

### Format Code

```bash
npm run format
```

## Customization

This template is designed to be a starting point. Feel free to:

- Add routing with [React Router](https://reactrouter.com/)
- Add state management with [Redux](https://redux.js.org/) or [Zustand](https://zustand-demo.pmnd.rs/)
- Add UI component libraries like [Material-UI](https://mui.com/) or [Ant Design](https://ant.design/)
- Add testing with [Vitest](https://vitest.dev/) and [React Testing Library](https://testing-library.com/react)
